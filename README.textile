h1. Watir Tests

This repository contains some tests to show how automate your tests using the awesome Watir.

h2. Installation

First of all, you need to install Watir.

If you are in Windows:

pre. gem install watir

If you are in Linux/Mac:

pre. gem install firewatir

After install Watir, clone the repository:

pre. git clone git@github.com:FabricioFFC/watir_tests.git

That's all. Enjoy! ;)
